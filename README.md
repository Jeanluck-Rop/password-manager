# Password Manager

A desktop **Password Manager** focused on securely storing and managing your passwords.
Built with **Rust**, **SQLCipher**, **JavaScript** with the **Tauri** tool.

## **How to Run the App** (No Cloning Code Required)

You **do not need to clone the repository** to use this application.

Thanks to Tauri, the app is distributed as native executables.

Currently **only Linux and Windows builds are available**.

### **Linux**

Go to the `app_pm/release` directory and download the installer that matches your distro.

#### **Ubuntu / Debian / Mint**

Download:
```
app_pm_0.1.0_amd64.deb
```

Install it with:

```
sudo dpkg -i app_pm_0.1.0_amd64.deb
```

#### **Fedora / Arch / RPM-based-distros**

Download the file:
```
app_pm-0.1.0-1.x86_64.rpm
```

Install it with:
```
sudo rpm -i app_pm-0.1.0-1.x86_64.rpm
```


### **Windows**

Go to `app_pm/release` and download:
```
app_pm.exe
```

Double-click the executable and allow execution if Windows shows a security prompt. The application is safe and does not contain malware.
